PROMPT 1: Yêu cầu sinh mã nguồn Spring Boot hoàn chỉnh
======================================================================
"Bạn hãy đóng vai một Senior Backend Developer (Java Spring Boot) với 10 năm kinh nghiệm.

Tôi cần bạn sinh mã nguồn cho module ĐĂNG KÝ TÀI KHOẢN CƠ BẢN của hệ thống eKYC dựa trên tài liệu SRS sau:

[DÁN PHẦN 3.1 CỦA SRS: API Register Specifications]

Yêu cầu kỹ thuật:
1. **Cấu trúc MVC đầy đủ**: 
   - Entity (JPA)
   - DTO (Request/Response)
   - Repository (Spring Data JPA)
   - Service (Interface + Impl)
   - Controller (REST API)

2. **Validation**:
   - @NotBlank, @Email, @Pattern
   - Custom Validator cho số CCCD 12 số
   - Custom Validator cho SĐT Việt Nam (10 số, đầu 03/05/07/08/09)

3. **Xử lý exception**: 
   - ValidationException
   - DuplicateResourceException
   - BusinessException

4. **JavaDoc và comment tiếng Việt** cho các method quan trọng

5. **Cấu hình**: 
   - application.yml với datasource, jpa, server config
   - pom.xml với dependencies

6. **Bonus**: Sinh sơ đồ kiến trúc Mermaid mô tả luồng Client -> Controller -> Service -> Database

Hãy sinh đầy đủ các file trong package: com.abcbank.ekyc"
